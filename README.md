# Team Task Manager

A full-stack task management application built using React, Node.js, Express, Prisma, and PostgreSQL.

## Features

- User Register
- User Login
- JWT Authentication
- Create Tasks
- View Tasks
- Update Task Status
- Delete Tasks
- Logout Functionality

## Tech Stack

### Frontend
- React
- Vite
- Axios

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL

## Deployment

- Frontend: Vercel
- Backend: Render
